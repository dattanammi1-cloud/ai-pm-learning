# AI Technical PM Foundation – Learning Repo

This repo tracks my journey to build a solid technical base for AI/ML Platform & LLM Product roles.

## Goals

* Get fluent with Python and Jupyter notebooks.
* Learn data wrangling, visualization, and basic statistics.
* Train classical ML models and a tiny neural net.
* Build intuition for LLMs (tokens, embeddings, attention, transformers).
* Ship small LLM product systems: RAG, tool-calling, and simple evals.
